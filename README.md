# 土豆优酷视频批量下载助手

土豆网和优酷网都是相当优秀的网站，其上有众多资源。而官网提供的下载工具，但是只能一个个下载，有时候需要下载某个豆单或者专辑的所有视频，一个个下载显然效率过低。这个工具只是调用了官方下载工具而已，不是地址嗅探工具。使用需要安装相应网站的下载工具。

## 项目已废弃
由于软件的实用性越来越低，加之原开发语言我基本已经不再使用，故本软件不再进行任何更新和修复。

推荐有需要的朋友使用[硕鼠] http://www.flvcd.com/

最终版本为2.0版本。
